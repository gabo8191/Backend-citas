## Instalación

1. Para instalar las dependencias, ejecute:

   ```bash
   npm install
   ```

## Ejecución

Para iniciar el servidor, ejecute:

```bash
npm run dev
```

## Almacenamiento de imágenes

Las imágenes se almacenarán en la carpeta `Uploads`, que se creará automáticamente si no existe.
